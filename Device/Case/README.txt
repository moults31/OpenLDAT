# box.stl 
* Print without supports, raft recommended for best results.  
* Optionally, glue a 22x22mm microscope glass on the bottom to protect the sensor
* Snap the OpenLDAT device in the holder, front side first

# top.stl
* Print as is with supports for best quality, or upside down without supports if you don't want to remove the supports, raft recommended for best results
* Remove the supports while still warm

# ledcover.stl (Optional)
* Print without supports and with skirt
* Snap inside the right hole in the top part
