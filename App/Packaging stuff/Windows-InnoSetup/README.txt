Requirements:
* Inno Setup
* Launch4j

Instructions:
* Place an x64 version of the JRE from OpenJDK inside the jre folder
* Copy the binaries from the dist folder of the OpenLDAT project inside the openldat folder
* Use launch4j to generate OpenLDAT.exe from launcher.xml, the exe will be automatically placed inside the openldat folder
* Open setup.iss with Inno Setup and build the setup pakcage
