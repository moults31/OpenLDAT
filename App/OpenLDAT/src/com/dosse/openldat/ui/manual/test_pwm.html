PWM / Strobing detection

This test is designed to detect the stability of the backlight of your display.
Most of the tests in this application will not work if the backlight is not stable.

This test takes approximately 10 seconds to complete.



What is PWM?

Some (cheap) panels do not have a dimmable backlight, so they "pretend" that they do by pulsing it on and off very quickly. By changing the duty cycle of this pulse (how long it stays on versus how long it stays off), different brightness levels can be achieved, and if the frequency is high enough, the human eye generally cannot see, but it causes eye strain and headaches over the long term.
If your display has this type of backlight, setting the brightness to maximum will usually disable it and allow you to run the tests and get accurate results.

What is Black Frame Insertion?

Black Frame Insertion is a common feature in modern high refresh rate gaming displays. It attempts to reduce ghosting caused by pixel response times by displaying a black frame between each "real" frame.
This feature should be turned off before running any test in this application.



Note on CRT displays

CRT displays have an extremely noticeable flickering caused by the way they refresh the screen with an electron beam. Using this type of display is not recommended with this application.



Other types of noise

Some older panels may have a dying backlight that flickers randomly. This will interfere with the tests.
Electromagnetic interference from power supplies, inverters, etc. may also get picked up by the device and cause inaccurate results. If too much interference is present, the test will be stopped.
