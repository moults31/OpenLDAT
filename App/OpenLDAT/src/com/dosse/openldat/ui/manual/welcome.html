Welcome to OpenLDAT!

OpenLDAT is the free and open source tool for measuring display and system latency metrics.

For the best results, follow these tips:

  • Run all tests in a dark room, with no lights around you
  • Keep the device perfectly still while the tests are running
  • Set the backlight brightness to 100%
  • Disable dynamic contrast, local dimming, ambient light sensors, power saving, etc.
  • Disable any kind of picture enhancement on your monitor and video driver
  • If you're using a TV, make sure it's set to Game Mode for lower latency
  • Disable any overlay or capture software such as MSI Afterburner, Xbox Game Bar, NVIDIA Shadowplay, etc.
  • Set the monitor to its native refresh rate
  • Always use full range RGB (8 bit or 10 bit)
  • Do not force VSync on or off from your video driver settings, let the application control it
  • Don't have multiple displays connected to this machine
  • Connect the display directly to the machine, avoid streaming over the network
